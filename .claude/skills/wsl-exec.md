# WSL Executor Skill

When Bash tool is unavailable (Windows limitation), use this workflow to execute operations in WSL.

## Steps

1. Write the Python script to `D:\仓库\工作日志\wsl_task.py` using the Write tool
2. Tell the user to execute in WSL:
   ```
   python3 /mnt/d/仓库/工作日志/wsl_task.py
   ```
3. After user confirms execution, read any output files generated by the script

## Common Tasks

### Read .docx files
Write a Python script that:
- Uses `python-docx` to open .docx files
- Extracts paragraphs/text
- Saves as `.txt` alongside the original file
- Then use Read tool to read the .txt

### Generate .docx files
Write a Python script that:
- Uses `python-docx` to create a new Document
- Formats content with proper headings, paragraphs
- Saves to specified path
